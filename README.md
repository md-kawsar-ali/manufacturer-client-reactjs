# Autima Pro MERN STACK Application

It's a Manufacturing web application! [Live Preview](http://autima-pro.web.app).

## Key Features

-   Login System
-   Registration System
-   Ordering System
-   Stripe Card Payment
-   Minimal Dashboard

## Technology used:

-   React JS
-   Node JS
-   Express JS
-   Mongodb
-   Firebase Authentication
-   DaisyUI and Tailwind CSS
-   JSON WEB TOKEN (JWT)
-   React Hook Form
-   React Query
-   SweetAlert2
-   Stripe
-   React Hot Toast
-   Swiper Slider
-   ImgBB (for Image Storage)
-   Dotenv etc...
